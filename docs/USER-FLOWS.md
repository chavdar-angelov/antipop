# CUSTOMER
Can:

* Sign-up
* Sign-in
* Browse products
* Browse brand profiles
* Add products to cart
* Place an order
* Add products to favourite

# BRAND OWNER
Can:

* Sign-up
* Sign-in
* Create a brand
* Add/edit products
** will elaborate later
* See list of orders
* Change order status
* Add/remove moderator
* Create campaigns
* Add/remove influencers from campaigns

# INFLUENCER
Can:

* Sign-up
* Sign-in
* Can post photos
* Can join campaigns
* Can tag products

# MODERATOR
Can:

* Sign-up
* Sign-in
* Do whatever the brand owner can except add/remove moderators

# SUPER ADMIN
* Can do everything (not in literal sense)
