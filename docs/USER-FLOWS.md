# OVERVIEW

## Public user can:
* Sign-up
* Sign-in
* Browse products
* Browse posts
* Add products to cart

## Customer can:

* Browse products
* Browse posts
* Browse categories
* Browse tags
* Add products to cart
* Checkout
* Like (favourite) products
* Like (favourite) posts
* Follow brands
* Follow influencers
* View purchases
* View favourites
* Return product(s)
* Create influence profile
* Create brand profile
* Update personal information
* Change password
* Delete profile

## Influencer can:

* Do everything a customer can!
* Add posts
* Tag products to posts
* Join campaigns
* View affilliate purchases

## Brand owner can:

* Do everything a customer can!
* Add/update products
* Add/update product variants
* Track orders
* Track returns
* Add/remove members (moderators)
* Create campaigns
* Approve/remove/invite influencers to campaigns
* Track affiliate purchases
